Connection(connection_uri: str, logSizeBytes: int=100*1024*1024)
    Connection represents a connection between the Hopps server and the
    MongoDB server.

    Parameters:
      * connection-uri: A string giving a MongoDB connection string for
        Hopps to connect to.
      * logSizeBytes: An optional parameter giving the maximum replication
        log size.
